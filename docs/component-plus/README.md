## 原生组件库

### 描述

web-plus-ui 组件库底层采用的 web-core-plus；样式采用 Element 组件库样式。目前组件库还在逐步完善中，基于 web-core-plus 工具包可以快速的定义 WebComponent，使用方式见 [web-core-plus](https://www.npmjs.com/package/@canyuegongzi/web-core-plus) 。
